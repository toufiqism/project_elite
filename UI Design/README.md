# Project Elite — Figma Exports

32 screen exports (16 unique screens × Light + Dark).

## Contents
- `light/` — light mode PNGs
- `dark/` — dark mode PNGs

## Screen list
1. **01-splash** — Splash / welcome
2. **02-onboarding-step1** — Personal info (name, age, body)
3. **03-onboarding-step2** — Goals (CA level, role, workout)
4. **04-onboarding-step3** — Discipline mode (Gentle / Balanced / Elite)
5. **05-home** — Dashboard with daily score, tasks, habits
6. **06-tasks** — Daily AI planner, all tasks grouped by time
7. **07-study** — CA study tracker, subjects list
8. **08-study-timer** — Focus session timer
9. **09-fitness** — Workout plan, weekly progress
10. **10-fitness-workout** — Active workout, exercises
11. **11-habits** — Habit tracker, monthly calendar
12. **12-prayer** — 5 daily prayers, dua, stats
13. **13-elite** — Elite Mode / character stats / radar
14. **14-stats** — Reports & analytics
15. **15-ai** — Elite AI chat assistant
16. **16-profile** — Profile + settings

## Notes
- Native render size: 402 × 874 (iPhone-class)
- Exported at: 248 × 540 (downscaled to fit capture viewport)
- For pixel-perfect Figma layers, use the **html.to.design** Figma plugin
  on the original `Project Elite.html` file — it imports as editable Figma layers.
